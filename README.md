# Deloitte Data Analytics: Factory Operational Analysis

## Project Overview
This project was completed as part of the **Deloitte Data Analytics Job Simulation**. I acted as a consultant to analyze large-scale datasets (160,000+ records) to identify operational trends and social impact metrics for a manufacturing client.

## Key Insights
- **Pay Equality Analysis:** Categorized factory pay structures into 'Fair', 'Unfair', and 'Highly Discriminative' tiers to assist executive decision-making.
- **Operational Trends:** Identified correlations between machine health and performance metrics to optimize factory output.
- **Data-Driven Strategy:** Provided actionable recommendations to address pay gaps and improve socio-economic impact.

## Tech Stack
- **Tableau:** For data visualization and interactive dashboarding.
- **Excel:** For initial data cleaning and categorical grouping.
